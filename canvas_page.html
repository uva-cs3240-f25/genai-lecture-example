CS 3240 — Responsible Use of GenAI in Coding

Date: Tuesday

Learning Outcomes

  • Explain why different GenAI models behave differently for coding.
  • Choose appropriate interfaces (browser, API, IDE plug‑ins) for a task.
  • Apply ethical considerations: licensing, attribution, privacy, security.
  • Cite AI-assisted code clearly and professionally.
  • Read and verify AI-generated code against stakeholder specs.

In-Class Demo (Django)

  1. Show the AI first draft of models/forms; ask students to spot what’s missing (ISBN, borrowing, validation).
  2. Switch to the corrected code. Run migrations and seed data. Borrow/return a book.
  3. Run tests: see weak test vs meaningful constraint test.
  4. Compare different tools on the “list users” view.
